# Windows

## Disable the annoying 'Ding'

Click:

1.  Control Panel
2.  Hardware and Sound
3.  Sound -> Change System Sounds
4.  Set 'Asterisk' and 'Default Beep' to 'None'

## Shortcuts

| Shortcut          | Effect         |
| ----------------- | -------------- |
| `Win + Shift + S` | Create Snippet |

## Fix VMMEM using too much CPU

```bash
wsl --shutdown
```
